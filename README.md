Desktop-приложение на Python для автоматического обновления Excel-отчёта по прогрессу MRDR. Реализовано чтение выгрузки из листа products, поиск нужных колонок по шаблонам названий, нормализация артикулов и дат, фильтрация данных за текущий и предыдущий год, удаление дублей и обновление существующего отчёта с добавлением новых колонок прогресса по дате выгрузки. Также настроено сохранение пользовательских параметров, графический интерфейс на Tkinter, форматирование Excel через openpyxl и ежедневный автозапуск через Windows Task Scheduler.
В работе выполнено:
Фильтрация данных по бизнес-логике: оставление только записей за текущий и предыдущий год.
Удаление дублей по артикулу с сохранением последней актуальной записи.
Автоматическое обновление существующего Excel-отчёта без создания нового файла.
Динамическое добавление новых колонок в отчёт.
Создание отдельных колонок прогресса по дате и времени изменения выгрузки.
Обновление существующих строк и добавление новых артикулов в отчёт.
Удаление устаревших строк из отчёта.
Работа с форматированием Excel: выравнивание, перенос текста, ширина колонок, высота строк, текстовый формат дат.
Сохранение пользовательских настроек в JSON-файл.
Поддержка ручного запуска через интерфейс и тихого автоматического режима через аргумент командной строки.
Создание .bat-файла для запуска обновления.
Настройка ежедневного автозапуска через Windows Task Scheduler.
Валидация путей к файлам, проверка существования отчёта и обработка ошибок.
